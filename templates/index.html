How to Use Personal AI Assistant

  • Select the file that you need information on.
  • Formulate a specific question regarding the selected bylaw.
  • Click the 'Submit' button to get your answer.
  • Note: Due to this being a prototype, please allow a few seconds for the response. Make sure to only click 'Submit' once to avoid duplicate responses.

Important Information

  • If you receive the message "Error Error Error" there are two reasons why:
  • 1) Please select a file on the left before asking a question. If you ask a question without selecting a file and press submit, you'll get an error.
  • 2) If you select a file, ask a question, and see 'Error Error Error,' it means there's an issue with how your question is formatted or the system is struggling to find the right information
  • If this happens, please note down your question and the content of the file you were inquiring about. Then, send this information to connor.pettepiece@hotmail.com.
